# E2E Basic Usage Example

This example demonstrates a basic end-to-end workflow of using the RAE Memory SDK to interact with a running RAE instance.

## Prerequisites

1.  **RAE API is running**: You must have a local instance of the RAE API running. See the [Getting Started Guide](../../docs/getting_started.md) for instructions.
2.  **SDK is installed**: You must have the `rae-memory-sdk` installed. You can install it from the local source:
    ```bash
    pip install ./sdk/python/rae_memory_sdk
    ```

## Run the Example

To run the example, navigate to this directory and run the `main.py` script:

```bash
cd examples/e2e_basic_usage
python main.py
```
